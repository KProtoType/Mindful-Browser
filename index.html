Mindful Browser

Take a mindful pause before visiting distracting websites

Why Use Mindful Browser?

We often find ourselves mindlessly opening social media tabs dozens of times per day out of habit, not intention. Mindful Browser adds a simple 1-minute pause before loading potentially distracting websites, giving you a chance to ask: "Do I really need to check this right now?"

This small intervention can make a big difference in how you use the internet, helping you reclaim your time and attention.

How It Works

Download & Install

Download the extension files from GitHub and install them in Chrome with just a few clicks - no coding required!

Choose Your Distracting Sites

Add patterns for websites that tend to distract you. The extension comes with Chess.com as a default, but you can add any sites you want.

Facebook YouTube Twitter/X Reddit Instagram TikTok

Experience the Pause

When you visit a distracting site, you'll see a blank page with a simple message for 60 seconds. After the pause, the website will load normally.

This brief delay is often enough to break the automatic habit and make a more mindful choice.

How to Install (No Technical Knowledge Required)

  1. Download the ZIP file using the button above
  2. Extract the ZIP file on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Turn on "Developer mode" in the top-right corner
  5. Click "Load unpacked" and select the extracted folder
  6. That's it! The extension is now installed
