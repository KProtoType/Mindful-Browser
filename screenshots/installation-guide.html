Visual Installation Guide for Mindful Browser

Download the Extension

Start by downloading the extension files from GitHub.

[Image showing the GitHub Code dropdown with Download ZIP option highlighted]
  1. Click the green "Code" button at the top of the GitHub page
  2. Select "Download ZIP" from the dropdown menu

Tip

The download will typically go to your Downloads folder, but you can choose a different location if prompted.

Extract the ZIP File

Find the downloaded ZIP file and extract its contents.

[Image showing right-click on ZIP file with Extract All option]
  1. Locate the "Mindful-Browser-main.zip" file in your Downloads folder
  2. Right-click the file and select "Extract All..." (Windows) or double-click (Mac)
  3. Choose a location where you want to extract the files
  4. Click "Extract"

Open Chrome Extensions Page

Open Google Chrome and navigate to the Extensions page.

[Image showing Chrome with extensions page open]
  1. Open Google Chrome
  2. Type chrome://extensions/ in the address bar and press Enter
  3. Or click on the three dots menu → Extensions → Manage Extensions

Enable Developer Mode

Turn on Developer Mode in Chrome Extensions.

[Image showing the Developer mode toggle in the top right]
  1. Look for the "Developer mode" toggle in the top-right corner of the Extensions page
  2. Click to turn it ON

Tip

Developer mode allows you to install extensions from outside the Chrome Web Store.

Load the Extension

Now load the extension files you extracted.

[Image showing the Load unpacked button and file selection]
  1. Click on the "Load unpacked" button that appears after enabling Developer mode
  2. Navigate to the folder where you extracted the Mindful-Browser files
  3. Select the "Mindful-Browser-main" folder (not any subfolder)
  4. Click "Select Folder" (Windows) or "Open" (Mac)

Success!

The Mindful Browser extension is now installed and ready to use.

[Image showing the installed extension in the Chrome toolbar]

By default, the extension will add a 1-minute delay when you visit Chess.com. You can customize this by clicking on the puzzle piece icon in the toolbar, finding Mindful Browser, and selecting Options.

Tip

If you want to add your own distracting websites, open the Options page and add patterns like *://*.facebook.com/* for Facebook or *://*.youtube.com/* for YouTube.
